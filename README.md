# ToDO_App
A demo ToDo App API built using the below mentioned keywords.

keywords:
- Python
- FastAPI
- RestfulAPI
- Motor (MongoDB driver)

Structure:
- name: name of the toDo
- desc: description of the toDo

How to run: 
- open mogoDB and create db `toDo`
- clone the repository
- compile toDo.py
- open: localhost:5000
